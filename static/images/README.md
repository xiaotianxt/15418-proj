# Images Directory

Place your images here. Images in this directory will be accessible at `/proj/images/your-image.png`

## Usage in Markdown

To use images in your content, reference them like this:

```markdown
![Alt text](/proj/images/your-image.png)
```

Or with HTML for more control:

```html
<img src="/proj/images/your-image.png" alt="Description" style="max-width: 100%;">
```

## Recommended Image Formats

- PNG for diagrams and screenshots
- JPG for photos
- SVG for vector graphics (logos, diagrams)
- WebP for optimized web images
