# Proposal Page Images

Place images specific to the proposal page here. These images will be accessible at `/proj/proposal/images/your-image.png`

## Usage in proposal/index.md

To use images in your proposal, reference them like this:

```markdown
![Alt text](images/your-image.png)
```

Or using absolute path:

```markdown
![Alt text](/proj/proposal/images/your-image.png)
```
